## Configuration

**Environment:**
- Environment variables loaded from `.env` file (gitignored)
- Key env vars:
  - `PUBLIC_IMGX_DOMAIN` - Imgix CDN domain (default: https://igcn-ws.imgix.net)
  - `PUBLIC_SITE_BASE_URL` - Site base URL
  - `TINA_CLIENT_ID` - TinaCMS client ID for authentication
  - `TINA_TOKEN` - TinaCMS API token

**Build:**
- Astro config: `astro.config.mjs`
  - Integrations: React, Sitemap, MDX
  - Markdown plugins: remark-gfm, remark-breaks, remark-emoji, remark-directive, unified-admonitions, remark-youtube (custom), rehype-slug, rehype-highlight
  - Custom plugins: `src/lib/remark-youtube.js`, `src/lib/unified-admonitions.js`
  - Manual chunks: react-vendor, search-vendor, utils-vendor
- Vite config: Integrated in `astro.config.mjs`
- TypeScript config: `tsconfig.json` (extends astro/tsconfigs/strict)
- Biome config: `biome.json` (linting and formatting)
- Vitest config: `vitest.config.ts`
- CSS config: `src/styles/globals.css` (Tailwind v4 + DaisyUI v5 configuration)

**No JavaScript Config for Tailwind:**
- Configuration is done entirely in CSS via `@import "tailwindcss"` and `@plugin "daisyui"`
- No `tailwind.config.js` or `postcss.config.cjs` files
